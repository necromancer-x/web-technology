HTML <head> Element Explained

This page explains what belongs inside the HTML head and why it matters in real-world web development.

What is the <head> element?

The head element contains metadata—information about the document such as its title, encoding, styles, scripts, and SEO-related data.

What belongs in the head?

  • Meta tags (charset, viewport, description)
  • Title of the document
  • Links to CSS files
  • JavaScript files
  • Icons and branding information

What does NOT belong in the head?

  • Headings
  • Paragraphs
  • Images
  • Forms
  • Any visible content

Key Rule to Remember

If something is shown on the page, it goes in the <body>. If it describes or configures the page, it goes in the <head>.
