HTML Media Elements

Learning how to use images, video, audio, and figure correctly

Image Element

The <img> tag is used to display images. The alt attribute is mandatory for accessibility.

Figure and Figcaption

<figure> groups media with its description, and <figcaption> explains it.

Video Element

The <video> element is used to embed video content. Using <source> allows multiple formats.

Audio Element

The <audio> tag is used for sound content like music or podcasts.
