CSS Box Model Masterclass

1. Content-Box (Default)

In content-box, the width and height apply only to the content.

content-box

300px + padding + border

Total width = 300px + 40px padding + 12px border

2. Border-Box (Recommended)

In border-box, width and height include padding and border.

border-box

Exactly 300px wide

Total width = 300px (everything included)

3. Inline vs Block Elements

Inline elements flow inside text: Inline element and do NOT respect width or height.

Block element
Takes full width
Respects width & height

4. Box Model Components

The CSS box model consists of:

  • Content - actual text or media
  • Padding - space inside the border
  • Border - edge around padding
  • Margin - space outside the box
